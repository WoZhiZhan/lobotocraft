# Lobotocraft

## 简介
Lobotocraft是以《脑叶公司》为主题的Minecraft Forge MOD，为游戏添加了还原的精神值，工作，出逃等大量机制。以及自定义实体和武器装备等内容。

**注意：** 本mod的设定独立于原作《脑叶公司》之外

## 特性

### 工作系统
- 玩家可以通过与异想体工作，获得对应的PE-BOX。解锁异想体信息和制作装备获取饰品
- 玩家可以通过不同类型的工作来提升自身属性和数值。或者晋级更高等级的属性
- 玩家可以通过工作来达到每日目标，进入下一天挑战更高的难度

### 武器系统
模组有独特的伤害系统，每把武器造成伤害的机制都不一样，一共分为4种伤害：红色，白色，黑色，蓝色

- **红色伤害**：会让玩家掉血
- **白色伤害**：会让玩家掉精神值
- **黑色伤害**：会让玩家同时损失血量和精神值
- **蓝色伤害**：会让玩家百分比扣血

部分武器还有特殊攻击提高观赏性和玩家的战斗性能

### 装备系统
玩家拥有伤害抗性机制，每套护甲的伤害抗性基本不相同。伤害抗性也分为：红色，白色，黑色，蓝色
- 对应伤害的抗性可以减少对应的伤害

MOD还提供大量饰品，专属饰品搭配专属装备可以触发套装效果，获得额外加成。

### 关于实体
MOD内可工作的实体统称为异想体，异想体在玩家工作时会根据工作类型，玩家等级来判定成功率，玩家工作失误会受到伤害惩罚，成功会累计

异想体有独特的计数器机制，部分异想体计数器归零后会触发对应的特殊事件，部分异想体计数器则为摆设没有作用。

除了异想体之外，模组内还会在野外生成一些特有怪物。为玩家添加游戏难度

### 独特UI
MOD内为玩家添加了不同的UI让玩家更好的了解信息

- **Shift + 右键** 未出逃的异想体：可以打开异想体的信息页面UI
- **右键** 异想体：可以打开异想体的工作UI
- 玩家背包在饰品栏位旁边有自己的信息页面按钮，玩家可以点击后跳转自己的数值页面
- 在玩家左上角可以看到抑制器的计数值

### 世界机制
玩家在除了公司维度之外的维度，都会缓慢降低自身抑制器的计数值，当计数值归零后，玩家会不断受到负面和伤害的影响

玩家恢复计数值需要对异想体进行工作。根据异想体不同的等级回复的数值也不同

### 等级机制
无论是武器，装备还是异想体等级都有区别。从小到大分别为：ZAYIN,TETH,HE,WAW,ALEPH

其中装备有独特的等级压制机制。

### 特殊道具
- **捕捉单元**：使玩家能够收容一只未出逃的异想体带在身上
- **传送装置**：从别的维度瞬间回到公司维度
- **TT2协议**：加速玩家的工作速度
- **再生反应堆**：为玩家提供回复效果
- **出逃方块**：异想体计数器归零后，如果会出逃则会瞬移到距离这个最近的方块上来

### 恐慌机制
玩家精神值空了之后会进入恐慌状态，恐慌状态目前分为两种：

1. **勇气恐慌**（血量大于精神值）
    - 玩家会不受控制，攻击周围的生物

2. **谨慎恐慌**（精神值大于等于血量）
    - 玩家无法移动，获得最高等级的失明和缓慢效果

**解除恐慌**：需要玩家受到来自其他玩家使用白色/黑色伤害武器的攻击（或者部分异想体也可以解除）
